# School Management System

## Overview
A full-stack School Management System that allows administrators to manage students, teachers, classes, attendance, and other school-related information through a user-friendly dashboard.

## Features
- Admin Login Authentication
- Student Management
- Teacher Management
- Class Management
- Dashboard Analytics
- Image Upload
- Responsive Design

## Tech Stack
Frontend:
- React.js
- Tailwind CSS
- Recharts
- Axios
- Lucide React

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer

## Installation

Frontend
npm install
npm run dev

Backend
npm install
npm run dev
